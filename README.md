# 🛡️ JowiAlchemyShield

A psychology-based trading dashboard designed to help traders manage **risk, emotion, and decision-making** before entering the market.

---

## 📌 Overview

JowiAlchemyShield is a frontend trading discipline tool that helps users:

- Track emotional state before trading  
- Calculate position size based on risk  
- View trading context (daily & weekly stats)  
- Follow a structured pre-trade checklist  
- Avoid revenge trading and emotional decisions  

---

## ⚙️ Features

### 🧠 Emotion Control System
- Mood slider (Calm → Neutral → High Risk Emotion)
- Trading warning messages based on emotional state

### 📊 Risk Calculator
- Account balance input
- Risk percentage selection (1%–3%)
- Entry & Stop Loss calculation
- Automatic position sizing
- Take Profit (2R reward system)

### 📅 Trading Context Panel
- Trades per day tracking
- Consecutive losses tracker
- Daily P&L
- Weekly P&L

### ⚠️ Drawdown Awareness
- Visual risk limits
- Warning thresholds (-5%, -10%)

### ☑️ Pre-Trade Checklist
- Stop loss confirmation
- Risk discipline check
- Anti-revenge trading reminder
