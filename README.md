# JowiAlchemyShield
The seatbelt for your trading account.
A pre-trade safety system for crypto traders.
What It Does
Mood Check — Slide how you feel. "Desperate" locks you out for 2 hours.
Risk Calculator — Enter balance, entry, stop. Get exact position size.
Pre-Trade Checklist — 3 boxes to check before you can trade.
Daily Stats — Tracks trades, losses, P&L for the day and week.
How to Use
Download JowiAlchemyShield_Final.html
Open in your browser before every trade
Complete all 4 sections
Click "Unlock Trading Terminal"
Why I Built This
I lost $24,000 trading on Bybit, Kucoin, and Binance. Not because I couldn't read charts — because I couldn't control myself. This tool forces me to pause, calculate, and stay disciplined.
Tech
Single HTML file
No server, no API, no tracking
Works offline
Data stays in your browser
